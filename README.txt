Project organization:

<<Version controlled by git>>
/  -- project files   
  SenM.sln  -- Visual Studio 2012 solution file
  SenM.vcxproj -- Real-time modeler
  genSCADA.vcxproj -- Synthetic SCADA data generator
  .gitignore -- git configurations of ignored files
  Doxyfile -- doxygen configurations
  README.txt -- this file

src/  -- C/C++ source code and compile-time dependencies  
   *.h, *.cpp -- SenM source files 
  SFMT_1.4/ -- Pseudo-random number generator [source files]
  rngs/ -- Random variable (distribution) generator [source files]
  iniParser_3.1 -- ini file parser [source files]
  EPANET-lt/ -- EPANET 2 with Lemon Tiger extension [source files]
  VTK_5.10.1/ -- Visualization toolkit [headers only]
  VTK_5.10.1_gen/ -- Visualization toolkit generated files [headers only]
  clapack_3.2.1/ -- LAPACK in C [headers only]

tests/ -- Tests and examples 
  Test*.vcxproj -- project files for tests
  Test*.cpp -- source code for tests
  data/  -- data or config files used by tests
    *.inp -- epanet network input file
    *.sql -- Mysql script for building test schema (table structures)
    *.ini -- data source config file
